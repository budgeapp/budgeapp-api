# Budge API

Backend API for the Budge app.

## Development

### Requirements

* Python 3.12
* [Poetry](https://python-poetry.org/)
* [poetry-plugin-dotenv](https://pypi.org/project/poetry-plugin-dotenv/)
* PostgreSQL 16
* GNU Make

### Setup

```bash
poetry install
poetry run make setupdb
```
